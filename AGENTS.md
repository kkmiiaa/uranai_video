# AGENTS ガイドライン

このリポジトリでエージェント（AI支援）を利用する際の共通ルールです。

## 基本方針

- 言語: 原則 日本語。
- 実行環境: Supabase は Docker、Flutter はホスト実行（iOS シミュレータ/macOS デスクトップ等）をメインとする。コンテナ利用は任意。
- コミット: 一つの変更単位ごとに自動コミットを行う。コミットメッセージは日本語を設定する。ホームディレクトリで `git add .` + `git commit -m "{変更メッセージ}"` の形式を取る

## 作業ルール

- 変更点の確認は `rg`/`cat` などでソースを読み、差分は PR/コミットで提示する。
- 画像やアセットの拡張子は勝手に変更しない（必要時は事前に合意）。
