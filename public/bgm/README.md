# BGM

`fortune.mp3` をこのディレクトリに配置してください。

## 推奨音源

**DOVA-SYNDROME** (https://dova-s.jp/) から著作権フリーのBGMをダウンロードしてください。

### おすすめの検索キーワード
- ヒーリング
- 癒し
- 幻想的
- スピリチュアル
- ピアノ 穏やか

### ライセンス確認事項
- 「商用利用可」のチェックを入れて検索
- ダウンロード後、ファイル名を `fortune.mp3` にリネームしてこのフォルダに配置

### GitHub Actionsでの使用
GitHub Actionsでレンダリングする場合、このファイルもリポジトリに含める必要があります。
音源のライセンスがリポジトリへの同梱を許可しているか確認してください。
DOVA-SYNDROMEの多くの楽曲は同梱可能ですが、個別に規約を確認してください。
